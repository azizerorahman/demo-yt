# 云南茶语 - Yunnan Tea House

A simple tea shop website with Chinese language support.

## Features

- Promotional banner with special offers
- Navigation header with search and cart icons
- Responsive design using Bootstrap 5
- Chinese font support

## How to Run

1. Open `index.html` in your web browser
2. That's it!

## Technologies

- HTML5
- Bootstrap 5
- Font Awesome icons
- Google Fonts (Chinese)
